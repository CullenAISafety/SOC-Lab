# SOC Reference Tools

This document lists essential tools and resources used in Security Operations Centers (SOC) for threat detection, investigation, and response.

---

## 1. Endpoint & Host Monitoring

- **Windows Event Viewer** – Native tool for monitoring Windows events and logs.  
- **Sysmon** – Advanced system monitoring for process creation, network connections, and file changes.  
- **Auditd (Linux)** – Kernel-level auditing for Linux hosts.  
- **OSSEC / Wazuh** – Host-based intrusion detection systems (HIDS).

---

## 2. Network Analysis

- **Wireshark** – Packet capture and network traffic analysis.  
- **Zeek (formerly Bro)** – Network monitoring and traffic logging framework.  
- **Snort / Suricata** – Network intrusion detection and prevention systems.  
- **tcpdump** – Command-line network packet capture tool.

---

## 3. Threat Intelligence & IOC Management

- **MISP** – Open-source threat intelligence platform for sharing IOCs.  
- **VirusTotal** – Check files, URLs, and IPs against known malware.  
- **OTX (AlienVault)** – Open threat exchange for indicator feeds.  
- **Cuckoo Sandbox** – Automated malware analysis.

---

## 4. Security Automation & Scripting

- **Python / PowerShell** – Automate alert processing, log parsing, and reporting.  
- **ELK Stack (Elasticsearch, Logstash, Kibana)** – Centralized log management and analytics.  
- **Splunk / Graylog** – Enterprise log analysis and alerting platforms.

---

## 5. Incident Response & Forensics

- **FTK / Autopsy** – Digital forensics tools for file and disk analysis.  
- **Volatility** – Memory forensics framework for analyzing RAM images.  
- **GRR Rapid Response** – Remote live forensics and monitoring.

---

## 6. References & Documentation

- MITRE ATT&CK Framework – Adversary tactics, techniques, and procedures (TTPs).  
- NIST Cybersecurity Framework – Guidelines for identifying and mitigating threats.  
- OWASP Top 10 – Security vulnerabilities for web applications.
